# slock

A simple screen locker utility for X written in C.

> All settings can be modified in [config.h](./config.h)

## Patches

- [capscolor](https://tools.suckless.org/slock/patches/capscolor)
- [dpms](https://tools.suckless.org/slock/patches/dpms)
- [message](https://tools.suckless.org/slock/patches/message)

## Options

| Option        | Action             |
| ------------- | ------------------ |
| `-m [string]` | set custom message |
